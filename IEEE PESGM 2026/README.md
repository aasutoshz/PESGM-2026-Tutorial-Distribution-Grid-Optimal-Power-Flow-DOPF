## Distribution-Grid-Optimal-Power-Flow-DOPF-Model
A comprehensive collection of Distribution Optimal Power Flow (DOPF) models for active distribution networks based on the branch-flow formulation. This repository provides mathematical formulations and implementations for solving optimal power flow problems in distribution networks.

The repository includes implementations of nonlinear programming (NLP), second-order cone programming (SOCP), semidefinite programming (SDP), and linear powerflow model (Lindist3flow) for solving multiphase unbalanced distribution optimal power flow problems.

## Features

- Branch-flow-based DOPF formulations for radial distribution networks
- Multiphase unbalanced power flow modeling
- NLP, SOCP, Lindist3flow and SDP optimization models
- IEEE test feeder implementations and case studies
- Convex relaxation and approximation techniques for scalable optimization
